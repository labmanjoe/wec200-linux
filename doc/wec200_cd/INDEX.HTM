Programs

Install Time Recorder Utility for Windows
This version is used to communicate and configure the WEC200-01 device.

Install Time Recorder Utility -Client for Windows
This version is used to access the Time Recorder Utility database remotely.

Install Device Explore for Windows
This version is used to browse and upgrade the internal software (firmware) of WEC200-01 terminal.

Install Tibbo IDE for Windows
This version is used to browse and upgrade the internal software (firmware) of WEC200-01 terminal. If your system is already installed Device Explore program, then it doesn't need to install this program.

Install Device Server for Windows (x86)
This version is used to browse and upgrade the internal software (firmware) of WEC200-00 terminal.

Install Device Server for Windows (x64)
This version is used to browse and upgrade the internal software (firmware) of WEC200-00 terminal.
